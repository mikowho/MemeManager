# MemeManager (表情包管理器) - 娱乐项目

一款基于 Android Jetpack Compose 开发的本地表情包管理工具。
本项目是 **niko** 与 **Gemini AI** 共同学习和探索 Android 开发的成果。
**严禁任何商业用途！** 仅供个人学习、交流和非营利性质的分享。

## ✨ 核心功能

*   **URL 直链下载**：输入 ZIP 链接，自动下载并解压，支持中文文件名解码。
*   **本地导入**：支持从手机文件系统导入 ZIP 表情包。
*   **智能重名处理**：导入时若检测到同名包，智能提示【覆盖】或【新建副本】（支持自定义命名）。
*   **分类管理**：自动根据文件夹生成标签页，支持**无限横向滑动**。
*   **自定义排序**：在管理界面支持表情包文件夹的 **置顶** 和 **上移** 操作，顺序永久保存。
*   **增删改查**：支持添加单张图片到表情包，长按删除单张图片，在管理页删除整个表情包文件夹。
*   **一键分享**：点击表情直接唤起自定义分享菜单，支持管理常用 App（如微信/QQ）。
*   **数据备份与恢复**：一键导出/导入 JSON 备份文件（含下载链接），换机/重装无忧。
*   **悬浮窗快捷入口**：支持桌面悬浮球，点击即达（需授权）。
*   **通知栏快捷启动**：常驻通知栏入口，极速启动。

## 🛠️ 技术栈

*   **语言**: Kotlin
*   **UI 框架**: Jetpack Compose (Material3)
*   **架构**: MVVM (Simplified)
*   **网络**: OkHttp
*   **图片加载**: Coil
*   **数据存储**: SharedPreferences + Gson
*   **文件操作**: Java Util Zip
*   **并发**: Kotlin Coroutines

## 🚀 如何安装

1.  在 Releases 页面下载最新版 `app-debug.apk`。
2.  安装到 Android 手机（需 Android 8.0+）。
3.  首次启动建议授予“通知权限”和“悬浮窗权限”以获得完整体验。
